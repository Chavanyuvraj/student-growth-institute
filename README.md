# 🎓 Student Growth Institute

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application designed to manage student admissions, courses, and admin operations.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 🔑 Forgot Password functionality
- 📚 Course Management System
- 📝 Student Admission Form
- 🧑‍💼 Admin Panel (Approve / Manage Admissions)
- 📱 Responsive UI for all devices

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- HTML5, CSS3, JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
